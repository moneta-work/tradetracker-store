A plugin that lets you create a store from the tradetracker productfeed

== Description ==

This plugin gives you the abillity to add a store to your wordpress based 
on a tradetracker productfeed. Tradetracker is an affiliate system that has 
the abbility to produce a product feed for you. So you can have a store
that brings in money without the hussle of owning a store. All you need to do
is choose a store connected on tradetracker and add it. 

Users of your blog will then see the products on your blog and when interested
they will be send to the store. When they buy an item you will get a percentage
of that sale.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php display_store_items(); ?>` in your templates
1. or use the template file that comes with the plugin
1. or use [display_store] in your page or post
